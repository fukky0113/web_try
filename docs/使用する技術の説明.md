## 使用する技術の説明

### Next.js（App Router）
- Reactベースのフルスタックフレームワーク。
- App Router: `app/` ディレクトリ配下のファイル構造がそのままURLルーティングになる。
- Server Component: サーバー側でレンダリング。データ取得に最適。
- Client Component: ブラウザ側で実行。`useState` などのインタラクションに使用。
- Server Actions: `"use server"` で定義した関数をフォームから直接呼び出し。APIルート不要。

### React
- UIを構築するためのJavaScriptライブラリ。
- コンポーネント単位で画面を分割して開発できる。
- `useState`: コンポーネント内の状態管理（タスク一覧、クイズの進行状態など）。
- `useTransition`: 非同期処理中のUI制御。
- `useRouter`: プログラムによるページ遷移やリフレッシュ。

### TypeScript
- JavaScriptに型定義を追加した言語。
- `type` キーワードでデータ構造を定義（C++のstructに相当）。
- コンパイル時に型チェックを行い、実行前にバグを検出。

### Tailwind CSS
- ユーティリティファーストのCSSフレームワーク。
- クラス名を組み合わせて素早くUIを構築できる。
- モダンなデザイン（グラデーション、ぼかし、シャドウなど）を簡潔に実装可能。
- レスポンシブ対応: `md:` プレフィックスで画面幅に応じたスタイル切り替え。

### HTML / JSX
- HTMLをベースにしたReact用の記法（JSX）。
- `class` → `className`、`for` → `htmlFor` に置き換えて記述。
- セマンティックなタグ（`header`, `main`, `section`）でアクセシビリティを確保。

### npm
- Node.jsのパッケージマネージャー。
- `package.json` で依存関係とスクリプトを管理。
- `npm run dev`: 開発サーバー起動。
- `npm run build`: 本番用ビルド作成。
