## index.html の解説

### 概要
- Tailwind CSS を CDN で読み込み、単一ファイルでモダンなランディングページを構成。
- セクションは「ヘッダー」「ヒーロー」「特徴一覧」の3つで構成。
- ダーク基調 + グラデーションの装飾で AI ツールらしい先進感を演出。

### 構成とポイント
- `header`
  - ロゴ、ナビゲーション、CTAボタンを横並びに配置。
  - `backdrop-blur` と半透明背景でガラス風の質感を演出。
- `section`（ヒーロー）
  - キャッチコピー、サブコピー、CTAボタン2つを配置。
  - 右側にダッシュボード風のカードを配置し、プロダクト感を強調。
  - 背景のグラデーションぼかしで奥行きを演出。
- `section`（特徴一覧）
  - セクション見出しと説明文を配置。
  - 3枚のカードで特徴を提示し、視線誘導を明確化。

### 使用クラス例
- レイアウト: `max-w-6xl`, `grid`, `md:grid-cols-2`, `gap-12`
- 配色: `bg-slate-950`, `text-slate-100`, `bg-indigo-500/30`
- 質感: `backdrop-blur`, `shadow-2xl`, `border-white/10`

### 変更しやすい箇所
- 文言: `h1`, `p`, `h2`, `h3` のテキスト
- CTA: ボタンのラベルとリンク先
- 配色: `bg-*`, `text-*` 系のクラス
